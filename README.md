# Mindora智元（Web UI）🚧

Mindora（Web UI）智能图片管理系统，AI驱动的图片管理解决方案前端实现，当前为开发初期阶段

## 🌟 规划特性

### 基础功能
- ⏰ 时间轴浏览（EXIF解析待实现）
- 🗺️ 地图模式（需要GPS数据支持）
- 🔍 混合搜索：
  - 自然语言描述搜索（规划中）
  - 以图搜图（开发中）
  - 视频语音搜索（仅限视频内容，待实现）

## 🛠️ 技术栈

### 前端
- Vue3 + Vite + Pinia

## 🚀 快速开始

### 环境要求
- Node.js 18+

### 安装步骤
```bash
# 克隆仓库
git clone https://github.com/<your-repo>.git

npm install

# 启动开发服务器
npm run dev
```
